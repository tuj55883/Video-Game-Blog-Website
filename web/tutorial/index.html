Tutorial Explanation

What is my component?

It is a vertical list of objects that only at first have a name, a picture of what it is, and a button to learn more. When the learn more button is clicked it launches a modal with more extensive information. All the information is organized by accordion folds so the user can access certain info that they want at a time. You can also search through the accordion folds for specific words in case you are looking for specific information. I made my component versatile so you could fille the list with anything.

How to use the component?

So the MakeInfoWindowList function takes in one parameter object that contains two thing (a name of the list, and the actual list/array of items you want to use). The item list is made by single objects that contain a (title, image URL, then an array of info). The array of info is made up of descriptors that act as indexes and values that act as contents. In theory you can change this array so your list can contain anything it wants. Then the MakeInfoWindowList will loop through the items and make a component for each and then put them into a big list that it returns that you can then put it into a div.
